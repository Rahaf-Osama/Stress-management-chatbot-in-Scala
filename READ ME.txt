Stress Management Chatbot
Project overview
The chatbot helps users manage stress by giving advice, breathing exercises, and activity recommendations based on the userĺs mood.
The chatbot can help with
* Anxiety 
* Panic
* Study stress
* Sleep problems
* Work stress 
* Overthinking

Main features 
* Chat with user using simple text input
* Suggest stress management techniques
* Recommend activities based on userĺs mood
* Store user preferences
* Save conversation history 
* Detect repeated questions 
* Show conversation summaries 
* Detect userĺs mood 
* Extract the most discussed topics

How the chatbot works 
The user chooses an option from the menu and enters a message.
The chatbot processes the input using keyword matching and pattern matching.
Depending on the detected keywords, the chatbot gives a suitable response or recommendation.
The chatbot also stores conversations history and user preferences for future interactions. 
Example:
User input: I feel anxious.
The chatbot responds with a breathing technique for anxiety.
The chatbot also stores this conversation history to use it later for recommendations and summaries.


Functional programming concepts used
1. Case classes
Used to represent data clearly:
* Activity
* Type
* InteractionEntry
* ConversationState
2. Pattern matching
Used for:
* Chatbot responses 
* Menu choices 
* Recommendation handling
3. Lists and maps
* Lists store activities and conversation history
* Maps store user preferences
4. Higher-Order functions
Used functions:
* Map
* Filter
* Foreach
* sortBy
* exists
* groupBy
5. Option type
Used in the recommendation engine to safely handle missing preferences.
6. Immutability 
The project uses immutable collections and updates states safely.

Installation 
1. Install scala on your device.
2. Open the project in VS code.
3. Add all Scala files to the same project.
4. Run the main Scala file to start the chatbot

How to run the program
Run the scala file 
The program will display a menu with different options
1. Chat with stress chatbot
2. Get activity recommendation
3. View conversation summary
4. View last conversation
5. View most discussed topics
6. Extract topics from the conversation 
7. Detect user mood
8. Play stress relief game
9. exit
choose a number and follow the instructions.

Architecture of the Solution
The project is divided into separate modules:
1. Core Chatbot Module
-Handles greetings, user input, response generation, and stress management techniques.

2. Recommendation Engine
-Handles user preferences and activity recommendations based on the userĺs mood.

3. Conversation Memory and context Tracker 
-Stores interactions, summarizes conversations, detects repeated questions, extract topics, and detects user mood.

This modular structure improves readability, organization, and maintainability. 

Bonus feature ľ Stress Relief Game
 The project also includes a bonus interactive game mode called the Stress Relief Challenge.
The game gives the user different stressful situations and multiple choices. The user must choose the healthiest response for each situation.
Examples of situations included:
* Exam stress 
* Panic and anxiety 
* Sleep problems
* Overthinking
* Work stress 
* Helping anxious friends
The game:
* Randomly selects questions using Random.shuffle
* Track the userĺs score
* Gives explanations after each answer
* Displays a final result message based on the score
This feature makes the chatbot more interactive and helps users learn healthy stress management habits in a simple and engaging way.

Conclusion 
This project is a simple stress management assistant created using Scala and functional programming concepts.
The chatbot gives supportive responses and recommendations while also keeping track of conversation history and user preferences.
